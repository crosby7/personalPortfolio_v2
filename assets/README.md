# Assets Folder

This folder contains all the assets for your personal portfolio website.

## Images Directory

- `profile.jpg` - Your profile picture for the hero section
- `about.jpg` - Image for the about section
- `project1.jpg` - Screenshot of your first project
- `project2.jpg` - Screenshot of your second project
- `project3.jpg` - Screenshot of your third project

## Recommended Image Sizes

- **Profile picture**: 400x400px (square, high quality)
- **About image**: 500x600px (portrait orientation)
- **Project images**: 600x400px (landscape orientation)

## File Formats

- Use JPEG for photos and complex images
- Use PNG for logos and images with transparency
- Use WebP for better compression (with fallbacks)

## Optimization Tips

- Compress images before uploading
- Use appropriate file sizes for web
- Consider using responsive image techniques
- Add alt text for accessibility

## Additional Assets

You can also add:

- Icons
- Logos
- Documents (resume, etc.)
- Videos or GIFs for project demos
